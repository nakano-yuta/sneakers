# 学習メモ
今回は、SMACSSベースでBEM記法を使ったコーディングを試してみた。<br>
SassのコンパイルにはPreprosを使っていて、slick.jsやinview.jsといったjQueryを使ったアニメーションを採用している。<br>
slick.jsのスライド間に余白を設けたい場合は、スライドの直下にdivタグを入れてそこにpaddigやmarginを指定するとうまくいく。<br>
